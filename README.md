# 🌐 Evolvit Club Website

A modern and responsive website for **Evolvit Club**, designed to promote innovation, collaboration, and skill development among students.

---

## 🚀 Features

* 🎯 Attractive and modern UI design
* 🧭 Smooth navigation experience
* ✨ Interactive sections like:

  * Join Now
  * Explore More
* 🎨 Styled with clean and visually appealing components

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
//* Firebase (for backend / hosting / authentication)(left)

---


## 📂 Project Structure

```
Evolvit-Website/
│── index.html

│── script.js
│── assets/
```


## 💡 Purpose

This project was created to:

* Showcase club activities and events
* Provide a platform for students to engage and collaborate
* Enhance frontend development skills

---

## ⚙️ How to Run Locally

1. Clone the repository:

```
git clone https://github.com/your-username/evolvit-website.git
```

2. Open the folder:

```
cd evolvit-website
```

3. Open `index.html` in your browser
